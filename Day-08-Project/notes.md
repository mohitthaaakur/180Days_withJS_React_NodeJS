# Expense Tracker Project

## What this project does
A console-based expense tracker that stores expenses as objects,
calculates totals, finds max/min expenses, converts currencies,
and searches expenses by name or category.

## Concepts practiced
- Constructor functions with new keyword
- Objects and object properties
- for...in loops
- String methods (substring, template literals)
- Math methods (Math.max, Math.min)
- Type conversion with +
- Arrow functions
- Conditional logic

## Features
- Add expenses with name, amount, category, date
- Calculate total expenses
- Find highest and lowest expense
- Calculate average expense
- Currency conversion (USD → INR, GBP)
- Selective conversion based on symbol input
- Search by category
- Search by expense name
