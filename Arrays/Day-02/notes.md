## Arrays - Day 2

### Exercises completed
1. Rotate Left - remove first element and push to end, repeated n times
2. Flatten 2D array - nested for..of loop pushing inner values to new array
3. Remove duplicates - indexOf returning -1 means value not yet in array

### Key learnings
- shift() takes no parameters, always removes first element
- for..in on array gives string indexes not values - use for..of or for loop
- indexOf returns -1 if value not found - useful for existence checks
- Simple one loop solution often better than complex multi-loop approach
