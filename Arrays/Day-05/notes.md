# Day 05 - Array Methods: map, sort, reverse

## Topics Covered
- map() - transform each element into new array
- sort() - default string sort, custom comparator (a - b) for numbers
- localeCompare() for sorting strings with special characters
- reverse() - reverses array in place
- Chaining sort and reverse for descending order
- Sorting array of objects by property

## Practice
- Mapped word lengths from string array
- Sorted numbers with custom comparator
- Sorted countries with localeCompare
- Sorted students array by marks descending
- Converted USD prices to INR with map
- Chained map and sort on names array
