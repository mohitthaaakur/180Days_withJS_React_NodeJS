## Arrays - Day 1 

### Topics covered
- Array declaration and basic methods
- push, pop, shift, unshift
- Loops on arrays (for, for..of)
- Array internals and references
- Multidimensional arrays
- toString conversion
- Array with mixed types

### Exercises completed
1. Array operations (styles - jazz, blues, rock)
2. Calling function in array context (this keyword)
3. Sum input numbers with prompt
4. Maximal subarray (Kadane's algorithm)
5. Find largest number without Math.max

### Key learnings
- Arrays are objects, assigned by reference not value
- +null converts to 0 — don't convert before validating
- Running sum reset to 0 when negative = efficient subarray logic
