# Day 06 - Strings

## Topics Covered
- Quotes (single, double, backticks)
- Special characters (\n, \t etc)
- String length
- Accessing characters with [] and at()
- Strings are immutable
- Changing case (toUpperCase, toLowerCase)
- Searching for a substring (includes, indexOf, lastIndexOf)
- Getting a substring (slice, substring)
- Comparing strings

## Exercises
1. ucFirst - uppercase first character of a string
2. checkSpam - case insensitive check for spam words
