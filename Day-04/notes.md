# Day 04 - Numbers in JavaScript

## Topics Covered
- Rounding (Math.round, Math.floor, Math.ceil, toFixed)
- Imprecise calculations with floating point numbers
- isFinite() and isNaN() for number validation
- parseInt() and parseFloat() for string to number conversion
- Math.random() for random numbers
- Math.max() and Math.min()

## Exercises (from js.info)
1. Sum numbers from visitor using Number(prompt)
2. Why 6.35.toFixed(1) != 6.3 - floating point precision
3. Repeat until input is a number using isFinite
4. Occasional infinite loop with floating point fix
5. Random number between min and max
6. Random integer between min and max
