# Day 03 - Symbol Type
## Topics Covered
- Creating symbols with Symbol()
- Symbols as hidden object properties
- Symbols skip for...in and Object.keys()
- No clash with same name string properties
- Global symbols with Symbol.for()
- Symbol.keyFor() to get name back from global symbol
- Regular symbols are always unique even with same description
